First-time setup

1 — Allow browser access
setx OLLAMA_ORIGINS "*"

Run once in Command Prompt or PowerShell.
macOS / Linux: add OLLAMA_ORIGINS="*" to your shell profile.

2 — Restart Ollama

Right-click the tray icon → Quit, then relaunch.
Or run ollama serve in a terminal.

3 — Verify

Click Test connection.
A 403 error means Ollama wasn't restarted after step 1.
